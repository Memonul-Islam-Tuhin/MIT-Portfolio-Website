# 🌐 MIT-Portfolio-Website

A simple **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**.
This project is designed to showcase a developer’s **skills, projects, and contact information** in a clean and responsive layout.

---

## 🚀 Features

* 👤 Personal Introduction Section
* 🛠 Skills Showcase
* 📂 Projects Section
* 📞 Contact Information
* 📱 Responsive Design
* ⚡ Lightweight and Fast

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📥 Clone the Repository

Open **Terminal / Command Prompt / Git Bash** and run the following command:

```bash
git clone https://github.com/Memonul-Islam-Tuhin/MIT-Portfolio-Website.git
```

---

## 📁 Navigate to the Project Folder

After cloning the repository, move into the project directory:

```bash
cd MIT-Portfolio-Website
```

---

## 💻 Open the Project in Code Editor

If you are using **Visual Studio Code**, run:

```bash
code .
```

If the command does not work:

1. Open **Visual Studio Code**
2. Click **File**
3. Click **Open Folder**
4. Select the **MIT-Portfolio-Website** folder

---

## ▶️ Run the Project

### Method 1: Open Directly in Browser

1. Go to the project folder
2. Find the file **index.html**
3. Double click it
4. The project will open in your browser

---

### Method 2: Using VS Code Live Server

1. Install **Live Server Extension** in VS Code
2. Right click on **index.html**
3. Click **Open with Live Server**
